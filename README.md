🚀 Employee Record System in C++ – My First Console-Based CRUD Project

Body:
I’m excited to share my latest C++ project – an Employee Record System that allows users to manage employee data efficiently through a simple console menu.

🔧 Features implemented:
✅ Add new employee records
✅ View all employee details
✅ Search employee by index
✅ Update existing employee info
✅ Delete employee records
✅ About section with team credits
✅ Clean formatted output using iomanip

📌 Tech used:
C++ | Arrays | Loops | Switch-case | Formatted I/O | Basic data validation

This project was developed as part of our college C++ assignment under the guidance of Deepa Ma’am, and our team put in 7+ hours to bring it to life.
